    Section 1 The basic pcf toolkit

    Subsection 1.1 What is \(\mathop{\mathrm{pcf}}(A)\) again?

    We assume that \(A\) is a progressive set of regular cardinals, that is \(|A|<\min(A)\text{.}\) We have defined \(\mathop{\mathrm{pcf}}(A)\) to be those cardinals \(\lambda\) that appear as the bounding number \(\mathfrak{b}(I)\) for some proper ideal \(I\) on \(\lambda\text{.}\) Using the density of exact sequences, we obtained an equivalent characterization in terms of true cofinalities.
    \begin{equation*} \mathop{\mathrm{pcf}}(A) = \{\mathop{\mathrm{tcf}}(I): I\text{ a proper ideal on $A$ for which $\mathop{\mathrm{tcf}}(I)$ exists}\}. \end{equation*}
    This can be re-formulated in terms of ultrafilters as well, yielding Shelah’s original definition:
    \begin{equation*} \mathop{\mathrm{pcf}}(A) = \left\{\mathop{\mathrm{cf}}\left(\prod A/ U\right): \text{$U$ an ultrafilter on $A$}\right\}. \end{equation*}
    We make extensive of use of the fact that \(\mathop{\mathrm{pcf}}(A)\) has a maximum value, which we denote \(\max\mathop{\mathrm{pcf}}(A)\text{.}\)

    Subsection 1.2 What about those ideals and generator things?

    In this part, we do a REALLY quick review of some of the work done earlier in the course.
    Let \(A\) be a progressive set of regular cardinals, and let \(I\) be a proper ideal on \(A\text{.}\)
    1. We know that the structure \(\prod A/ I\) has a bounding number \(\mathfrak{b}(I)\text{,}\) and that this bounding number is an infinite regular cardinal. By definition, \(\mathfrak{b}(I)\) is the largest cardinal \(\lambda\) such that the reduced product \(\prod A/ I\) is \(\lambda\)-directed.
    2. There is a set \(B_I\) in \(I^+\) such that
      • \(\mathop{\mathrm{tcf}}\left(\prod A/ I\restriction B_I\right)=\lambda\text{,}\) and
      • if \(B\subseteq A\) and \(\mathop{\mathrm{tcf}}(\left(\prod A/ I\restriction B\right)=\lambda\text{,}\) then \(B\subseteq B_I\) modulo \(I\text{.}\)
      Thus, he collection of \(B\in I^+\) for which the product \(\prod B/I\) has true cofinality \(\lambda\) is non-empty, and has a maximal (modulo \(I\)) member which we denote \(B_I\text{.}\) This is one of the key uses of our assumption that \(|A|<\min(A)\text{,}\) as we use the density of exact upper bounds to produce \(B_I\text{.}\)

    Subsection 1.3 Compactness for Generators

    Lemma 1. Suppose \(X\subseteq A\text{.}\) Then \(\max\mathop{\mathrm{pcf}}(X)\) is the least \(\lambda\) such that \(X\in I_{\leq\lambda}\text{.}\)
    Proof. If \(X\in I_{\leq\lambda}\) then \(\max\mathop{\mathrm{pcf}}(X)\leq\lambda\text{,}\) as there is a finite \(Y\subseteq\mathop{\mathrm{pcf}}(A)\cap\lambda^+\) such that
    \begin{equation*} X\subseteq\bigcup_{\theta\in Y} B_\theta[A]. \end{equation*}
    Now if \(\lambda\) is the least element of \(\mathop{\mathrm{pcf}}(A)\) with \(X\in I_{\leq\lambda}\) then we know a couple of things. First, \(X\) is NOT in \(I_{<\lambda}\text{.}\) Second,
    \begin{equation*} X\setminus B_{\lambda}[A]\in I_{<\lambda} \end{equation*}
    because \(B_{\lambda}[A]\) generates \(I_{\leq\lambda}\) over \(I_{<\lambda}\text{.}\) Putting these facts together tells us
    \begin{equation*} \lambda = \mathop{\mathrm{tcf}}\left(\prod X/ I_{<\lambda}\restriction X\right) \end{equation*}
    and therefore \(\lambda\) is in \(\mathop{\mathrm{pcf}}(X)\text{.}\) ◻
    Notice that the above proof shows us the following useful fact:
    Corollary 2. Suppose \(X\subseteq A\) and \(\lambda=\max\mathop{\mathrm{pcf}}(X)\text{.}\) Then
    \begin{equation*} \max\mathop{\mathrm{pcf}}\left(X\setminus B_\lambda[A]\right)<\lambda. \end{equation*}
    Theorem 1 (Compactness Property for Generators). If \(X\subseteq A\) then there is a finite subset \(\{\lambda_i:i<n\}\) of \(\mathop{\mathrm{pcf}}(X)\) such that
    \begin{equation*} X\subseteq B_{\lambda_0}[A]\cup\dots\cup B_{\lambda_{n-1}}[A]. \end{equation*}
    Proof. Let \(X_0 = X\text{,}\) and let \(\lambda_0 = \max\mathop{\mathrm{pcf}}(X)\text{.}\) We define sets \(X_i\) and cardinals \(\lambda_i\) by the following recipe:
    If \(X_i=\emptyset\text{,}\) then the construction halts. If \(X_i\) is non-empty, define
    \begin{equation*} \lambda_i = \max\mathop{\mathrm{pcf}}(X_i), \end{equation*}
    and
    \begin{equation*} X_{i+1} = X_i\setminus B_{\lambda_i}[A]. \end{equation*}
    By the preceding lemma, we know that if \(\lambda_{i+1}\) is defined, then \(\lambda_{i+1}<\lambda_i\text{.}\) Thus, we must reach a stage \(n\) where \(X_n = \emptyset\text{,}\) and the conclusion follows. ◻
    If \(X\subseteq A\) and \(A\) is a progressive set of regular cardinals, then how do the ideals \(I_{<\lambda}[A]\) (constructed for \(A\)) and the corresponding ideals \(I_{<\lambda}[X]\) (constructed for \(X\)) relate?

    Subsection 1.4 The ideals \(J_{<\lambda}[A]\)

    Definition 3. Suppose \(A\) is a progressive set of regular cardinals. For any infinite cardinal \(\lambda\text{,}\) we define
    \begin{equation*} J_{<\lambda}[A]: = \{X\subseteq A: \max\mathop{\mathrm{pcf}}(A)<\lambda\}. \end{equation*}
    Proposition 4. Let \(A\) be a progressive set of regular cardinals, and let \(\lambda\) be an infinite cardinal.
    1. \(J_{<\lambda}[A]\) is an ideal on \(A\text{,}\) and is a proper ideal if and only if \(\lambda\leq\max\mathop{\mathrm{pcf}}(A)\text{.}\)
    2. \(J_{<\lambda}\subsetneqq J_{\leq\lambda}\) if and only if \(\lambda\in\mathop{\mathrm{pcf}}(A)\text{.}\)
    3. If \(\lambda\in\mathop{\mathrm{pcf}}(A)\) then \(J_{<\lambda}[A]= I_{<\lambda}\text{.}\)
    Proof. We prove only the third statement, so assume \(\lambda\in\mathop{\mathrm{pcf}}(A)\text{.}\) Since
    \begin{equation*} \max\mathop{\mathrm{pcf}}(B_\theta[A])=\theta \end{equation*}
    for each \(\theta\in\mathop{\mathrm{pcf}}(A)\text{,}\) we know
    \begin{equation*} I_{<\lambda}\subseteq J_{<\lambda}[A]. \end{equation*}
    For the other direction, suppose \(\max\mathop{\mathrm{pcf}}(X)<\lambda\text{.}\) Apply the Compactness Property for Generators and find \(\lambda_0>\dots>\lambda_{n-1}\) in \(\mathop{\mathrm{pcf}}(X)\) such that
    \begin{equation*} X\subseteq \bigcup_{i<n}B_{\lambda_i}[A]. \end{equation*}
    Since each \(\lambda_i\) is less than \(\lambda\text{,}\) we know \(X\in I_{<\lambda}\text{.}\) ◻

    Subsection 1.5 Cofinality sequences

    For \(\lambda\in\mathop{\mathrm{pcf}}(A)\text{,}\) fix a sequence \(\bar{f}^\lambda=\langle f^\lambda_\alpha:\alpha<\lambda\rangle\) of functions in \(\prod A\) such that
    We call \(\bar{f}^\lambda\) a cofinality sequence for \(\lambda\text{,}\) and the entire collection \(\langle \bar{f}^\lambda:\lambda\in\mathop{\mathrm{pcf}}(A)\rangle\) will be referred to as a cofinality array for \(\mathop{\mathrm{pcf}}(A)\text{.}\)
    For now, we will use these sequences in the following way:
    Lemma 5. Suppose \(X\subseteq A\text{,}\) and let \(\lambda = \max\mathop{\mathrm{pcf}}(X)\text{.}\) Given a function \(g\in\prod A\text{,}\) there is an \(\alpha<\lambda\) such that
    \begin{equation*} \{a\in X: f^\lambda_\alpha(a)\leq g(a)\}\in J_{<\lambda}[A]. \end{equation*}
    In English, if \(\lambda = \max\mathop{\mathrm{pcf}}(X)\) then any function in \(\prod X\) can be dominated by some \(f^\lambda_\alpha\text{,}\) except for a set of exceptions that has smaller \(\max\mathop{\mathrm{pcf}}\text{.}\)
    Proof. Since \(\lambda = \max\mathop{\mathrm{pcf}}(X)\text{,}\) we know that \(X\) is a subset of \(B_\lambda[A]\) modulo \(J_{<\lambda}[A]\text{,}\) as \(B_\lambda[A]\) generates \(J_{\leq\lambda}[A]\) over \(J_{<\lambda}[A]\text{.}\) Thus,
    \begin{equation*} X\setminus B_{\lambda}[A]\in J_{<\lambda}[A]. \end{equation*}
    Given \(g\in\prod A\text{,}\) choose \(\alpha<\lambda\) such that
    \begin{equation*} g<f^\lambda_\alpha\mod J_{<\lambda}[A]\restriction B_\lambda, \end{equation*}
    so for \(J_{<\lambda}[A]\)-almost all \(a\in B_\lambda[A]\text{,}\) we have \(g(a)< f^\lambda_\alpha(a)\text{.}\) Then for this choice of \(\alpha\text{,}\)
    \begin{equation*} \{a\in X: f^\lambda_\alpha(a)\leq g(a)\}\in J_{<\lambda}[A]. \end{equation*}

    Subsection 1.6 What do we mean by “pcf structure”?

    Given a progressive set of regular cardinals \(A\text{,}\) we will often speak about the \(\mathop{\mathrm{pcf}}\)-structure of \(A\), and by this we mean the collection of ideals \(J_{<\lambda}[A]\text{,}\) the sequences of generators \(B_\lambda[A]\) for \(\lambda\in\mathop{\mathrm{pcf}}(A)\text{,}\) and some choice of cofinality sequences \(\bar{f}^\lambda\) for \(\lambda\) in \(\mathop{\mathrm{pcf}}(A)\text{.}\)
    Note that the ideals are unique and the generator \(B_\lambda[A]\) is unique modulo the ideal \(J_{<\lambda}[A]\text{.}\) The sequence of ideals is also completely determined by the sequence of generators \(\langle B_\lambda[A]:\lambda\in\mathop{\mathrm{pcf}}(A)\rangle\text{,}\) as \(J_{<\lambda}[A]\) is just the ideal on \(A\) generated by the family \(\{B_\theta[A]:\theta\in\mathop{\mathrm{pcf}}(A)\cap\lambda\}\text{.}\)
    Note that by the density of exact upper bounds, we can assume that \(\bar{f}^\lambda\) has an exact upper bound \(f^*_\lambda\) mod \(J_{<\lambda}[A]\) such that \(f_\lambda^*(a)\leq a\) for all \(a\in A\text{.}\) If this is the case, then we can recover a generator \(B^*_\lambda\) from \(\bar{f}^\lambda\) by defining
    \begin{equation*} B^*_\lambda:=\{a\in A: f^*_\lambda(a) = a\}. \end{equation*}
    We will pursue this in more detail a little later on.

    Subsection 1.7 Application: the cofinality of \(\prod A\)

    Theorem 2. Suppose \(A\) is a progressive set of regular cardinals. Then
    \begin{equation*} \mathop{\mathrm{cf}}\left(\prod A, <\right) = \max\mathop{\mathrm{pcf}}(A). \end{equation*}
    Proof. Let \(\langle B_\lambda[A]:\lambda\in\mathop{\mathrm{pcf}}(A)\rangle\) and \(\langle\bar{f}^\lambda:\lambda\in\mathop{\mathrm{pcf}}(A)\rangle\) generate the pcf structure of \(A\text{.}\)
    Given \(g\in\prod A\) we claim there is a finite subset \(\{\lambda_i:i<n\}\) of \(\mathop{\mathrm{pcf}}(A)\) and ordinals \(\alpha_i<\lambda_i\) for \(i<n\) such that
    \begin{equation*} g<\max\{f^{\lambda_i}_{\alpha_i}:i<n\} \end{equation*}
    in \(\prod A\text{.}\) Clearly this is sufficient, as there are at most \(\max\mathop{\mathrm{pcf}}(A)\) possibilities for the functions as on the right hand side.
    We work by induction, similar to how we proved Theorem [cross-reference to target(s) "thm1" missing or not unique], except this time we use Lemma [cross-reference to target(s) "lem4" missing or not unique] to fuel the argument. The idea is as follows: let \(\lambda=\max\mathop{\mathrm{pcf}}(A)\text{,}\) and choose \(\alpha<\lambda\) such that \(g<_{J_{<\lambda}[A]} f^\lambda_\alpha\text{.}\) This means \(f^\lambda_\alpha\) does what we want, except for a set of exceptions that is in \(J_{<\lambda}[A]\text{,}\) that is,
    \begin{equation*} \max\mathop{\mathrm{pcf}}\{a\in A: f^\lambda_\alpha(a)\leq g(a)\}<\lambda. \end{equation*}
    If this set is non-empty, then we continue by applying Lemma [cross-reference to target(s) "lem4" missing or not unique].
    In more precise language, we will implement a construction that attempts to define sets \(A_i\text{,}\) cardinals \(\lambda_i\text{,}\) and ordinals \(\alpha_i<\lambda_i\) for each \(i<\omega\text{.}\)
    We begin by setting \(A_0 = A\) and \(\lambda_0=\max\mathop{\mathrm{pcf}}(A_0)\text{.}\) We let \(\alpha_0<\lambda_0\) be the least ordinal such that
    \begin{equation*} \{a\in A_0 : f^{\lambda_0}_{\alpha_0}\leq g(a)\}\in J_{<_{\lambda_0}}. \end{equation*}
    Given \(A_i\text{,}\) if \(A_i\) is non-empty then set \(\lambda_i = \max\mathop{\mathrm{pcf}}(A_i)\) and let \(\alpha_i<\lambda_i\) be least such that
    \begin{equation*} \{a\in A_i: f^{\lambda_i}_{\alpha_i}(a)\leq g(a)\}\in J_{<_{\lambda_i}}. \end{equation*}
    Such an \(\alpha_i\) exists by way of Lemma [cross-reference to target(s) "lem4" missing or not unique]. Finally, define
    \begin{equation*} A_{i+1} = \{a\in A_i: f^{\lambda_i}_{\alpha_i}(a)\leq g(a)\}. \end{equation*}
    Note that our construction guarantees that if \(A_{i+1}\) is non-empty, then \(\lambda_{i+1}\) is strictly less than \(\lambda_i\text{.}\) Thus, there must be some least \(n\) where \(A_n\) is empty. This means that
    \begin{equation*} (\forall a\in A)(\exists i< n)\left[ g(a)<f^{\lambda_i}_{\alpha_i}(a)\right], \end{equation*}
    and so we are done. ◻
