    Section 5 No large intervals

    Theorem 11. If \(A\) is a progressive set of regular cardinals, then \(\mathop{\mathrm{pcf}}(A)\) cannot contain an interval of cardinality \(|A|^{+4}\text{.}\)
    Proof. Suppose by way of contradiction that \(\mathop{\mathrm{pcf}}(A)\) contains an interval \(X\) of regular cardinals of cardinality \(|A|^{+4}\text{.}\) We may assume that \(X\) consists entirely of successor ordinals, and that \(X\) is of order-type \(|A|^+\text{.}\)
    If we let \(\kappa\) denote \(|A|\text{,}\) then we can assume
    \begin{equation*} X = \{\eta_\alpha^+:\alpha<\kappa^{+4}\} \end{equation*}
    where \(\{\eta_\alpha:\alpha<\kappa^{+4}\}\) is an interval of cardinals that we want to think of as a copy of \(\kappa^{+4}\text{.}\)
    We will define a continuous strictly increasing function
    \begin{equation*} \Phi:\kappa^{+3}\rightarrow \kappa^{+4} \end{equation*}
    by induction on \(\alpha<\kappa^{+3}\text{.}\) We will also have an associated function
    \begin{equation*} \Phi^+:\kappa^{+3}\rightarrow X \end{equation*}
    defined by
    \begin{equation*} \Phi^+(\alpha) = \eta^+_{\Phi(\alpha)}. \end{equation*}
    In order to define the function \(\Phi\text{,}\) we let \(S\) be the stationary subset \(S^{\kappa^{+3}}_{\kappa^+}\) of \(\kappa^{+3}\text{,}\) and fix a club-guessing sequence \(\langle C_\delta:\delta\in S\rangle\) such that \(\mathop{\mathrm{otp}}(C_\delta)=\kappa^+\) for each \(\delta\in S\text{.}\) We assume that \(C_\delta\) is a set of limit ordinals, as this causes no harm and simplifies notation a little later on.
    We start by setting \(\Phi(0)=0\text{,}\) and for limit \(\alpha\) we know that \(\Phi(\alpha)\) must be the supremum of earlier values.
    Now how do we define \(\Phi(\alpha+1)\) if we know \(\Phi(\beta)\) for all \(\beta\leq\alpha\text{?}\)
    For each \(\delta\in S\) we will ask a question, which we refer to as “The \(C_\delta\) question for \(\alpha\)”: we ask
    This is a question that makes sense, as \(C_\delta\cap \alpha\) is a subset of \(\alpha\text{,}\) and \(\Phi^+[C_\delta\cap\alpha]\) is a subset of \(X\text{.}\) The answer may be either yes or no. If the answer is no, then this is because the corresponding \(\max\mathop{\mathrm{pcf}}\) is larger than \(\sup(X)\text{.}\)
    There are only \(\kappa^{+3}\) ordinals \(\delta\) to consider, so we can choose \(\Phi(\alpha+1)<\kappa^{+4}\) so large that for any \(\delta\in S\text{,}\)
    \begin{equation*} \max\mathop{\mathrm{pcf}}\left(\Phi^+[C_\delta\cap\alpha]\right)<\sup(B)\Longrightarrow \max\mathop{\mathrm{pcf}}\left(\Phi^+[C_\delta\cap\alpha]\right)<\Phi^+(\alpha+1). \end{equation*}
    Thus, if the \(C_\delta\) question for \(\alpha\) has a “yes” answer, then \(\Phi(\alpha+1)\) is large enough so that \(\Phi^+(\alpha+1)\) is greater than \(\max\mathop{\mathrm{pcf}}(\Phi^+[C_\delta\cap\alpha])\text{.}\)
    This construction defines the function \(\Phi:\kappa^{+3}\rightarrow\kappa^{+4}\text{,}\) and we will get our contradiction by looking at the behavior of sets of the form \(\Phi^+[C_\delta]\) for \(\delta\in S\text{.}\) Our construction is designed so that the following claim must hold:
    Claim 1. For each \(\delta\in S\text{,}\)
    \begin{equation*} \max\mathop{\mathrm{pcf}}\left(\Phi^+[C_\delta]\right)>\sup(X). \end{equation*}
    Proof. If this fails for \(\delta\text{,}\) then for each \(\alpha\) the \(C_\delta\) question for \(\alpha\) was answered with “yes”. Let us now suppose that \(\alpha\) is in \(\mathop{\mathrm{nacc}}(C_\delta)\text{,}\) and let \(\beta\sup(C_\delta\cap\alpha)\text{.}\) Note that since \(C_\delta\) consists of limit ordinals, we know \(\beta+2<\alpha\text{.}\)
    The \(C_\delta\) question for \(\beta+1\) had the answer “yes”, and so our construction guarantees
    \begin{equation*} \max\mathop{\mathrm{pcf}}\left(\Phi^+[C_\delta\cap\beta+1]\right)<\Phi(\beta+2)<\Phi^+(\alpha). \end{equation*}
    But \(C_\delta\cap\beta+1 = C_\delta\cap\alpha\text{,}\) and so for \(\alpha\in \mathop{\mathrm{nacc}}(C_\delta)\) we have
    \begin{equation*} \max\mathop{\mathrm{pcf}}\left(\Phi^+[C_\delta\cap\alpha]\right)<\Phi^+(\alpha). \end{equation*}
    Now we define
    \begin{equation*} B:=\Phi^+[\mathop{\mathrm{nacc}}(C_\delta)] \end{equation*}
    and note
    \begin{equation*} \theta\in B\Longrightarrow \max\mathop{\mathrm{pcf}}(B\cap\theta)<\theta, \end{equation*}
    which is a contradiction as \(|B|=|A|^+\text{.}\) ◻
    To finish the proof, let us define
    \begin{equation*} \mu = \sup\mathop{\mathrm{ran}}(\Phi^+) \end{equation*}
    Then \(\mu\) is a singular cardinal of cofinality \(\kappa^{+3}\text{,}\) and thus there is a closed unbounded \(C\subseteq\mu\) of order-type \(\kappa^{+3}\) such that
    \begin{equation*} \max\mathop{\mathrm{pcf}}(\{\theta^+:\theta\in C\})=\mu^+<\sup(X). \end{equation*}
    Since \(\{\eta_{\Phi(\alpha)}:\alpha<\kappa^{+3}\}\) enumerates a closed unbounded subset of \(\mu\text{,}\) we can find a closed unbounded set \(E\subseteq\kappa^{+3}\) such that
    \begin{equation*} \{\eta_{\Phi(\alpha)}:\alpha\in E\}\subseteq C. \end{equation*}
    Now choose \(\delta\in S\) such that \(C_\delta\subseteq E\text{.}\) Then
    \begin{equation*} \Phi^+[C_\delta]\subseteq\{\theta^+:\theta\in C\} \end{equation*}
    and therefore we have found a \(\delta\in S\) for which
    \begin{equation*} \max\mathop{\mathrm{pcf}}\left(\Phi^+[C_\delta]\right)\leq\mu^+<\sup(B). \end{equation*}
    This contradicts the previous claim. ◻
